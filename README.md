# 💰 多预算记账本 (自用)

一个简洁优雅的移动端多预算记账应用，支持多分类预算管理、周期性自动重置、实时支出追踪等功能。

## ✨ 功能特性

### 🎯 核心功能
- **多预算管理** - 同时管理多个独立预算（餐饮、交通、购物等）
- **智能周期重置** - 支持日/月/年周期，自动重置预算额度
- **实时记账** - 快速记录支出，即时更新剩余额度
- **数据可视化** - 进度条直观显示预算使用情况

### 📊 预算管理
- 自定义预算名称、图标、颜色
- 三种周期类型：每日/每月/每年
- 预算超支预警（红色警示）
- 一键编辑/删除预算

### 📝 记账功能
- 选择预算分类记账
- 支出明细列表
- 实时计算总支出
- 时间戳自动记录

### 💾 数据存储
- 本地 LocalStorage 持久化存储
- 数据自动保存，刷新不丢失
- 隐私安全，数据不上传云端

## 🚀 快速开始

### 方式一：直接打开
1. 下载 `rjrj.html` 文件

3. 双击用浏览器打开即可使用

### 方式二：部署到服务器
将文件上传至任意 Web 服务器或静态托管平台（GitHub Pages、Vercel、Netlify 等）

## 📱 使用指南

### 创建预算
1. 点击右上角 **+** 按钮
2. 设置预算名称（如：餐饮、交通）
3. 输入预算金额
4. 选择周期类型（每日/每月/每年）
5. 选择图标和颜色
6. 点击保存

### 记录支出
1. 点击预算卡片选中要记账的分类
2. 在"记一笔"区域输入支出项目
3. 输入金额，点击确认记账
4. 查看"支出明细"确认记录成功

### 预算周期说明
- **每日预算**：每天 0 点自动重置（如：每日餐费 50 元）
- **每月预算**：每月 1 日自动重置（如：月交通费 300 元）
- **每年预算**：每年 1 月 1 日自动重置（如：年旅游基金 5000 元）

## 🔒 隐私说明

- 所有数据存储在浏览器本地
- 不会上传任何数据到服务器
- 清除浏览器数据会导致记录丢失
- 建议定期导出重要数据（新增文件导出导入功能）
